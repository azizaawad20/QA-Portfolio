# Login Bug Reports

## BUG-001

**Title:**
Login button remains disabled after entering valid credentials

**Steps to Reproduce:**
1. Open Login page
2. Enter valid username
3. Enter valid password

**Expected Result:**
Login button becomes active.

**Actual Result:**
Login button remains disabled.

**Severity:** High

**Priority:** High

---

## BUG-002

**Title:**
Password field accepts more than maximum allowed characters

**Steps to Reproduce:**
1. Open Login page
2. Enter password longer than allowed limit

**Expected Result:**
Extra characters are not accepted.

**Actual Result:**
System accepts all entered characters.
